• API URL

    API URL

    Request Method : POST

    URL: https://filstest.stridewebstudio.com/api/createTransaction

    Authorization Header: Bearer Token (token: Merchant Secret Key)

  • Request Body

    Request Body

    Body Format: Form Data

    api_key : Merchant API Key

    total : Postive decimal value

    mode : 1,2
    1 : get the donation amount from the Total entered
    2 : Create transaction from the Total entered

    transaction_type : 1,2
    1 : One Time
    2 : Round Up

    channel_id : 1,2,3
    1 : Cash
    2 : Online
    3 : QR

    Sample Body Format- Mode 1 : Form Data

    total : 0.95
    mode : 1
    api_key : AMYVZT5-MGPMWVT-QMFV1B9-Q87CCHA
    transaction_type : 1
    channel_id : 2

    Sample Body Format- Mode 2 : Form Data

    total : 150
    mode : 2
    api_key : AMYVZT5-MGPMWVT-QMFV1B9-Q87CCHA
    transaction_type : 1
    channel_id : 2
